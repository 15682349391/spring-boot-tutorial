# spring-boot-data-jpa

## 使用方法

运行以下命令可以启动应用

```
cd codes/data/spring-boot-data-jpa
mvn clean package -DskipTests=true
java -jar -Dspring.profiles.active=dev target\spring-boot-data-jpa.jar
```
